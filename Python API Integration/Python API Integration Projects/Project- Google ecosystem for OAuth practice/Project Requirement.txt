📄 Project: Google Sheets Data Sync API Integration
🧾 Client Scenario
A small business uses Google Sheets to track leads. They want automation to sync this data with an internal system via APIs.
Name on Github - “Automated data synchronization system between Google Sheets and external APIs with retry handling, logging, and deployment support”
🎯 Objective
Build a Python-based integration service that:
    Authenticates using OAuth 2.0 - DONE
    Reads data from Google Sheets - DONE
    Processes/transforms it - DONE
    Sends it to a mock external API - DONE
    Handles failures, retries, and logging

🧩 Functional Requirements

1. OAuth 2.0 Authentication - DONE
Use OAuth 2.0 to:
    Authenticate user via Google
    Store access + refresh tokens
    Automatically refresh expired tokens
👉 Constraint: DONE
    No manual token copy-paste after first run
    Must persist tokens (file/db)

2. Read Data from Google Sheets - DONE
Use Google Sheets API
    Read rows from a sheet like:
Name	Email	    Status
A	    a@mail.com  NEW

👉 Requirements: - NOT DONE
    Handle pagination (large sheets)
    Validate schema (no missing fields)

3. Data Transformation - DONE
Before sending:
Convert:
    Name → full_name
    Email → lowercase
Filter:
    Only Status = NEW

4. Send to External API - DONE

Simulate sending to:
https://reqres.in/api/users  OR mock server

Use:
    POST requests
    JSON payload
        Responsibilities:
            send_single_lead(payload)
            send_batch_leads(list_of_payloads)
            retry logic
            error logging
👉 Requirements: - DONE
    Retry on failure (429, 500)
    Timeout handling
    Proper error logging

5. Logging & Observability - DONE
Log:
    API requests/responses - DONE
    Failures - DONE
    Token refresh events - DONE
Use structured logging (JSON preferred) - DONE

6. CLI Interface - LEFT (need to learn subParsers for this)
Command examples:
    python app.py sync
    python app.py sync --dry-run
    python app.py status

You must handle: - DONE

    API rate limiting → exponential backoff
    Partial failure (some rows fail)
    Empty sheet
    Invalid email format
