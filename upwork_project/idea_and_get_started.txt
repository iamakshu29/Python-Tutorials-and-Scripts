⚠️ The Big Catch — Upwork API Access
This is where most people get stuck:

Upwork's API is not publicly open — you need to apply for API access
They are quite restrictive and mainly approve agencies/businesses
Personal developer access is hard to get approved
Their API also doesn't officially support posting proposals (it's limited)
